title Рисунок 2b — TO-BE: загрузка видео и модерация

start
:1. Пользователь авторизуется (анонимный токен);
:2. Заполняет форму: название, описание, категория, теги;
:3. Загружает видеофайл;
:4. Система ставит файл в очередь обработки;
:5. Выполняются асинхронные задачи:
   - конвертация в разные битрейты;
   - извлечение метаданных (длительность, разрешение);
   - генерация превью (скриншотов);
   - автоматическое создание субтитров (whisper);
:6. Все метаданные сохраняются в PostgreSQL;
:7. Видео отправляется на модерацию;
if (Модератор одобряет?) then (да)
  :8a. Видео публикуется в общем каталоге;
  :8b. Автор получает уведомление;
else (нет)
  :8c. Видео отклоняется с указанием причины;
  :8d. Автор может загрузить исправленную версию;
endif
stop
